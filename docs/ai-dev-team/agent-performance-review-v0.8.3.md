# Agent Performance Review (v0.8.3)

## 概要
Agent Performance Review は、Gemini, Claude, こさめPM の各役割が、どれだけ効果的にプロジェクトに貢献したかを評価し、改善に繋げるための仕組みである。

## 評価項目
- **Gemini (Bulk Generator)**: 大量生成タスクの成功率、修正不要率。
- **Claude (Repair Agent)**: 補修タスクの成功率、再発防止策の提案数。
- **こさめPM (Decision Support)**: 判断材料の整理品質、承認の迅速さ。
- **Cloud Shell (Verification)**: 検証の安定性、エラーの検知精度。

## 目的
- エージェントの得手不得手を可視化し、適切な役割分担を行う。
- ネタ要素ではなく、実運用におけるボトルネック（詰まりどころ）を特定する。
- 継続的なプロンプト改善や運用ルールの洗練に役立てる。
