# Gemini Bulk Work Result Report (v0.5.5)

## 報告テンプレート
Gemini は大量作業完了後、以下の形式で報告を行う。

1. **作業概要**: 何を目的とした作業か。
2. **作成・更新ファイル数**: 合計数と主要なパス。
3. **未完了・スキップ項目**: 境界線に触れたため、あるいはエラーでスキップした項目。
4. **次アクションへの申し送り**: 誰が、どのコマンドで検証すべきか。
