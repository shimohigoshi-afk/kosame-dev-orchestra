# KOSAME Dev Orchestra Product Idea Discovery Pack v61.0.0

## 概要

作る前にアイデアの筋が良いかを整理するpackです。
誰の課題か、SNSで伝わるか、競合がいるか、差別化できるかを評価します。

## 評価観点

- 1秒で何のアプリか伝わるか (`oneSecondUnderstandingScore` 0〜10)
- ターゲットユーザーの悩みが明確か
- 競合の存在を「需要の証拠」として活用できるか
- じゅんやさんの営業力・現場知見が活かせるか
- 海外バズを日本向けにローカライズできるか

## 安全設計

- `dryRun: true` / `humanApprovalRequired: true`
- 実広告・実LP公開・実SNS投稿なし

## 使用方法

```bash
npm run pm-agent:product-idea-discovery
npm run smoke:product-idea-discovery
```
