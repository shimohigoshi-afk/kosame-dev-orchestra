# KOSAME Dev Orchestra v6.2.0 Project Template Applicator

## Purpose

This release provides a template generator that applies the KOSAME Dev Orchestra standard structure — docs, tools, smoke, fixtures, approval gates — to any new or existing repository.

## Standard Structure Applied

- Directories: docs/ai-dev-team, tools, smoke, fixtures
- Files: release record, operation standard, pack tool, smoke test, fixture
- Approval gates: commit, push, tag, deploy (all require じゅんやさん approval)
- Scripts: verify, smoke:all placeholders

## Release Value

v6.2.0 allows new KOSAME projects to be bootstrapped with the full Dev Orchestra governance structure in a single operation, ensuring consistency across all product lines.
