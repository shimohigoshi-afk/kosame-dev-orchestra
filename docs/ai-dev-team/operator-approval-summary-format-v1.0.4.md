# Operator Approval Summary Format v1.0.4

## Summary Structure

```markdown
### 📋 Approval Request: [Action Name]

**Risk Level**: [Low/Medium/High/Critical]
**Requested By**: [Agent Name]

**Summary of Changes**:
- [Change 1]
- [Change 2]

**Why this is needed**:
- [Reasoning]

**Risk Factors**:
- ⚠️ [Risk 1]
- ⚠️ [Risk 2]

**Suggested Options**:
1. ✅ **Approve**: Proceed immediately.
2. ⏸️ **Hold**: Need more investigation.
3. 🤖 **Send to Claude**: Requires technical fixing.
4. ❌ **Reject**: Stop this approach.
```
