# Operator Console Screen Layout (v0.9.1)

## 画面構成
- **Header**: プロジェクト名、現在のバージョン、最終更新日時。
- **Left Sidebar**: ナビゲーション（Dashboard, Approvals, Handoff, History）。
- **Main Content**:
  - **Upper Row**: 状態カード（Status, Agent, Risk, Approval Count）。
  - **Middle Row**: 次のアクション（大きなボタンまたはテキスト）。
  - **Lower Row**: 詳細リスト（承認待ちリスト、最近の変更履歴）。
- **Right Sidebar**: エージェント状況（Gemini, Claude の現在地）。

## 配色設計
- Background: ダークモード推奨（#121212）。
- Success: #4CAF50 (Green)
- Warning: #FFC107 (Amber)
- Error: #F44336 (Red)
- Info: #2196F3 (Blue)
