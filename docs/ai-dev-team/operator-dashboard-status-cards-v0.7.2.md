# Operator Dashboard Status Cards (v0.7.2)

## 標準カードセット
1. **Version Card**: `currentVersion` を表示。
2. **Verify Card**: 最新の `npm run verify` 結果を表示。
3. **Actions Card**: 推奨される `nextAction` を表示。
4. **Approval Card**: `pendingApproval` の件数を表示。
5. **Risk Card**: `riskLevel` を表示。
6. **Agent Card**: `activeAgent` を表示。

## カスタムカード
各フェーズ（Deploy, Release, BugFix）に応じて追加のカードを表示可能にする。
