# Operator Runbook (v0.6.0)

## 概要
Operator Runbook は、KOSAME Dev Orchestra を用いた開発・運用における標準的な「手順書」である。自動化しきれない部分や、人間が介在すべきポイントを明確にする。

## 主要ランブック一覧
- **RB-001: 朝の稼働確認**: システムの健康状態チェック
- **RB-002: 新機能実装フロー**: 設計からデプロイまで
- **RB-003: バグ修復フロー**: エラー検知からパッチ適用まで
- **RB-004: 緊急リリース**: P0 インシデント対応
- **RB-005: お引越し・環境構築**: 新プロジェクトへの流用

## 運用ルール
1. 実行したランブック ID をセッションログに記録する。
2. 手順に不備が見つかった場合は即座にドキュメントを更新する。
